K7TTY Software Library

The K7TTY Software Library was the factory support site for Baudot RTTY software written by Bill Bytheway K7TTY, original software author for RTTY.COM and ITTY. His programs — including RTTYMailer and the Bytheway Software Development Lab (BSDL) suite — were written specifically for RTTY and the Baudot code.

The original library at k7tty.com is currently unavailable. Work is underway to locate and archive Bill’s software here on RTTY.COM. If you have copies of BSDL software or know of a mirror, please contact W2TTY via the GreenKeys list.

RTTYMailer v16 — the current version for submitting content to ITTY channels — is available on the ITTY page.
